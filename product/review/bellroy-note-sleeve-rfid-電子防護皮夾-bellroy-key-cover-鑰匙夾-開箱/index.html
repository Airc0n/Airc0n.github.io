Bellroy Note Sleeve RFID 電子防護皮夾/ Bellroy Key Cover 鑰匙夾 開箱

2020-01-23 Pageviews: --

過新年，沒有換新衣，但皮包已經脫線，這是一個 porter 的皮包，是我當時的女朋友(現在的太太)送我的生日禮物，

但不敵我屁股的摧殘，開始推線，皮革也開始劣化，才動心起念想要換掉這個皮包，

加上最近常常被 youtube 演算法推薦的 EDC 系列   (Every Day Carry) 所吸引，

我很喜歡的一位 youtuber （Peter McKinnon）做了一部影片來推薦了一些 EDC Wallets

Bellroy 是其中一個推薦的品牌之一，就是那隻貓頭鷹 logo，是一個澳洲品牌 （點我看官網）

原以為需要跨洋過海的訂購，後來才發現台灣竟然有代理商，且價格並沒有因此而翻倍，

台灣代理商 摩登工廠（點我看摩登工廠官網），商家位於 基隆中山區，

不但有 2020 年的新款式，也提供了三年保固，於是我挑了兩樣喜歡的 產品

一個是 Bellroy Note Sleeve RFID 電子防護皮夾 ，另一個是 Bellroy Key Cover 鑰匙夾

就這樣在小年夜的前一天來到了我手上，不得不說，商家的包裝真的蠻好的

商品本身有牛皮紙袋包覆，還有發票 以及 他們所提供的 公司貨保固卡

精裝的牛皮紙袋上還壓印有 Bellroy 的貓頭鷹 logo ，蠻精緻的

打開後我選購的皮包還有一層薄薄的紙張包覆

跳快打開牛皮紙袋的過程，快速來分享我的心得

Bellroy Note Sleeve 經典直式 RFID 電子防護皮夾 (款式為 RFID-Java 內裏棕)

Java 色 真的很難拍出來，是一種深咖啡色的皮革，剛拿出袋子，一定要用力給他聞一下皮革的味道，

但老實說，皮革味道並不明顯，不像棒球手套那樣，可以聞到濃郁的皮革味，

皮革拿起來的觸感蠻滑溜的，也比我原本的皮包還小，

其實我個人蠻喜歡另外一款的 貓頭鷹 鍍鉻 logo (應該是吧) 鑲嵌再皮包外，那個質感超棒，在搭配咖啡色的皮革，

套句對岸的說法，這樣的配置，真香～

大約丈量了一下皮包的大小，寬大約 9cm 高約 10 公分，蠻小的

跟我的舊愛放在一起，看得出來 porter 已經留有了不少歲月的痕跡，

把我的東西移轉到新的 bellroy 皮包後，乾淨了許多，中間還留有一個小皮革可以讓你快速把不大常使用的卡片一次抽出，

常用的三張卡片就放在外側，隨時可以抽取，

放鈔票的地方有鋼印指出 Made In India

另外這個皮包有個特點就是他標榜可以 有 RFID 的保護，

因為現在的電子支付或資料交換很常透過 NFC 感應 或著 RFID 感應，曾聽聞皮包放在褲子口袋，被不肖人士透過儀器竊取卡片資料，

而這款皮包具有阻擋感應的功能，這邊有位網友拿了這款皮包做了實際的實驗(3:54 秒開始)，

放上 1000 和 100 的寬度比例，新台幣 1000 塊錢的寬度應該是這個皮包剛好可以放置的極限

真的蠻薄的，我想應該是我把所有卡片都堆疊一起了

這張小卡是一開始放在皮包裡面的使用說明吧ＸＤ

Bellroy Key Cover 鑰匙夾 (款式為 Plus-Cocoa)

接著要介紹的是這款鑰匙夾，既然都買了皮夾，平常在口袋磨來磨去的鑰匙，也是需要「隔絕」一下

於是就買了一個鑰匙的家，交棒一下，換鑰匙夾出場

裡面還貼心的附贈了假鑰匙，讓你知道放起來的樣子，有一點很特別的是

鑰匙夾的邊緣是有吸磁的，所以可以讓上下透過磁力吸附在一起

老實說這面車線 我沒有很愛….

不過打開看到了貓頭鷹搭配棕色的皮革，還是覺得，真香～

內部的皮革車線和 串接鑰匙孔的繩子，用料都算不錯，但穿鑰匙的時候，真的不是很好用，有可能是我的鑰匙太厚了…

至於 plus 款的高度約 9 公分，寬度約 4.5 公分

結論

兩個商品我很喜歡，但美中不足的是，我是 1/19 透過蝦皮訂購並加價透過黑貓寄出，商家則是 1/21 寄出商品，

而我實際收到商品的時間是 1/22，但保固卡上卻寫著 1/20 ，建議商家可以把運送時間「彈性」的送給客戶，

感謝 摩登工廠，讓我在過年前有新皮包和新鑰匙夾可以用，

如果看完文章對於 bellroy 有興趣，可以到他們的
